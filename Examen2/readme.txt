En el siguiente vienen dos archivos .py una solución incluye el login completo web, el cual debe instalar primeramente flask, luego pyodbc para el manejo de la base de datos con SQL server.
Se debe ejecutar e ingresar /login el mismo desplegara un acceso donde debe ingresar el nombre de usuario y la contraseña de la base de datos en sql server llamada examen2 y de la tabla usuarios.
Una vez realizada el ingreso correcto nos mostrará el menú principal.
Posterior a esto tenemos otra app que es en consola la cual consulta y modifica los datos de los usuarios desde la una base de datos en posgres, se adjunta script de ambas bases de datos.
